// Sass will preserve the format of your color... unless it is in something other than the keyword, hex, or rgba formats. If you want to absolute format, then you have to turn into string For example : - In scss variable file // $mercury-grey: #{'rgba(230, 230, 230, 1)'}; Input .class { color: $mercury-grey; } Output:- .class { color: rgba(230, 230, 230, 1); }
